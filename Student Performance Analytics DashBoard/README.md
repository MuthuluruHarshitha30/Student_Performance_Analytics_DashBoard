# 🎓 Student Performance Analytics Dashboard

## 📌 Project Overview

The Student Performance Analytics Dashboard is a Data Analytics project developed using Python, Pandas, Matplotlib, and Power BI. The project analyzes 5,000 student records to identify academic, behavioral, and demographic trends and presents insights through interactive dashboards.

---

## 🚀 Objectives

* Analyze student academic performance data
* Perform data cleaning and validation
* Generate meaningful KPIs
* Conduct Exploratory Data Analysis (EDA)
* Visualize trends and patterns
* Build interactive Power BI dashboards for decision-making

---

## 🛠️ Technologies Used

* Python
* Pandas
* Matplotlib
* Power BI

---

## 📊 Dataset Information

* Records: 5,000 Students
* Attributes: 23 Columns
* Categories:

  * Academic Performance
  * Attendance
  * Study Habits
  * Stress Levels
  * Sleep Patterns
  * Demographic Information

---

## 🔍 Data Analytics Workflow

### 1. Data Cleaning

* Checked missing values
* Identified duplicate records
* Handled null values in Parent Education Level

### 2. Exploratory Data Analysis (EDA)

* Grade Distribution Analysis
* Attendance Analysis
* Academic Performance Analysis
* Correlation Analysis

### 3. KPI Generation

* Total Students
* Average Total Score
* Average Attendance
* Average Study Hours
* Average Sleep Hours
* Average Stress Level
* Pass Percentage

### 4. Data Visualization

* Grade Distribution
* Attendance vs Total Score
* Study Hours vs Total Score
* Stress Level vs Total Score
* Sleep Hours vs Total Score
* Correlation Heatmap

### 5. Power BI Dashboard

* Executive Dashboard
* Academic Performance Analysis
* Behavioral & Demographic Insights

---

## 📈 Key Insights

* Majority of students fall within Grade C and Grade D categories.
* Average attendance rate is approximately 75%.
* Average total score is approximately 72.
* Behavioral and demographic factors were explored to understand their relationship with academic performance.
* Interactive dashboards provide quick access to student performance trends.
